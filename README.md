# future-dashboard

Thin read-only frontend for the `future` reporting exports.

## Data source

This repo expects JSON snapshots generated by the sibling `future` repo:

```bash
cd ../future
python scripts/export_dashboard_data.py
cp dashboard_exports/*.json ../future-dashboard/public/data/
```

## Run

```bash
npm install
npm run dev
```

## Scope

- overview cards
- sales timeseries
- top events
- top venues
- data quality snapshot

No provider API access lives here.
